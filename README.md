# NestEgg Landingpage

Scrollbasierte Premium-FinTech-Landingpage für den NestEgg Businessplan.

## Dateien

- `index.html` – Seitenstruktur und Businessplan-Inhalte
- `styles.css` – Premium-FinTech-Designsystem, Responsive Layout, Animation Styles
- `script.js` – Scrollspy, Intro-Reveal, Tabs, Charts, Command Palette, Chat-Demo
- `README.md` – Nutzungshinweise

## Nutzung

1. ZIP entpacken.
2. `index.html` im Browser öffnen.
3. Für den vollen Wow-Effekt sollte eine Internetverbindung bestehen, da GSAP/ScrollTrigger und Chart.js per CDN geladen werden.

## Eingebaute Funktionen

- Fullscreen-Intro mit Reveal in die eigentliche Landingpage
- Sticky Side Navigation mit Scrollspy
- Scroll-Progressbar und Mini-Map
- Cinematic Hero mit Chat-Demo und KPI-Widgets
- Problem-, Solution-, Product-, Market-, Business-Model-, Competition-, Trust-, Financials- und Roadmap-Sections
- Chart.js Visualisierungen für Markt, Finanzplanung und Mittelverwendung
- Interaktive Tabs für Geschäftsmodell und Personas
- Businessplan-Drawer
- Command Palette über `Ctrl/Cmd + K`
- Responsive Layout mit Mobile-Menü
- Reduced-Motion-Fallback für Nutzer mit reduzierter Bewegungspräferenz

## Deployment

Die Seite ist statisch und kann direkt mit GitHub Pages, Vercel, Netlify oder lokal im Browser genutzt werden.
